# CLDR for ruby on rails

This is a small rails engine which loads cldr jsons.

Following scripts are loaded:

* lib/js/cldr/supplemental/likelySubtags.json
* lib/js/cldr/main/de/numbers.json
* lib/js/cldr/main/de/currencies.json
* lib/js/cldr/main/de/ca-gregorian.json
* lib/js/cldr/supplemental/timeData.json
* lib/js/cldr/supplemental/weekData.json
* lib/js/cldr/supplemental/currencyData.json
* lib/js/cldr/supplemental/numberingSystems.json
